The anteriormost two scales in Basilemys have been variously interpreted. Earlier works (Riggs, 1906; Hay, 1908, 1910; Lang- ston, 1956) interpreted the most anterior of these (set 1) as the intergulars, and the second set as the gulars. If this interpretation is correct, then the condition in B. sinuosa and B. praeclara, with unreduced set 2 scales meeting in the mid-line, would have to be considered primitive, and the very reduced set 2 scales in B. variolosa and B. nobilis would be derived.
